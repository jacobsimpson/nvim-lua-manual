    Neovim APIs

    Neovim APIs

    There are a variety of APIs available for scripting Neovim using Lua.

    • vim.split or vim.gsplit
    • vim.trim
    • vim.api.nvim_*
    • Built in functions are available as vim.fn.*
    • See :help lua-stdlib for more details.
